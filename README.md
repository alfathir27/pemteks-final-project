## Preparing
### API KEY
- `./01. Scraping/Scraping YouTube.ipynb` -> API Key YouTube (dari Google Cloud Console)
- `./03. Labeling/process.py` -> API Key Gemini (dari Google AI Studio atau Vertex AI)

### Cookies
- `./01. Scraping/scrappingx.py` -> Cookies AUTH_TOKEN dan CT0 (dari browser yang membuka tab X dan sudah login)
